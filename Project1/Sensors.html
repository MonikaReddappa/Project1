Sensors

There are two types of sensors implemented in these applications: those that detect objects on the floor and those that detect potholes and bumps. 

Smartphone cameras identify objects around the user through image processing techniques. After this data has been gathered, the application on the smartphone processes it through an obstacle detection algorithm and provides vibration or audio feedback to the user. In addition, the user can take a picture of the environment to gain a better understanding of the surroundings. Smartphone cameras are used for this purpose. The picture taken from the smartphone is then processed to recognize the object in the image using various image-processing techniques such as edge detection or texture detection algorithms.

 
In the calibration process, we use accelerometers and ultrasonic sensors to store the distance to the ground from a smartphone in a database. When using this device, whenever the distance stored in the database exceeds the current value of the sensor, we refer to it as a bump. If it exceeds the value stored in the database, then we refer to it as a pothole.
